## Описание пункта 7

7. Создать расширение Frm-класса Project проекта Bas (Bas.Frm.Ref.Project) и выполнить следующее:
  7.1. Переименовать вкладку "Основное" в "Информация";
  7.2. При очищении поля "Дата начала" очищать поле "Дата окончания", если оно заполнено;
  7.3. При ручном вводе в поле "Код" реализовать проверку, что новое значение должно содержать только цифры, иначе выдавать ошибку;
  7.4. Сделать обязательными для заполнения поля "Название" и "Дата начала". При сохранении выдавать ошибку, если поле не заполнено;
  7.5. Добавить в шапку формы новую кнопку "Заполнить по шаблону", при нажатии заполнять запись проекта следующими значениями:
    - поле Название = "Тестовый проект";
    - поле Дата начала = установить дату сегодня;
    - поле Дата окончания = поле Дата начала + 30 дней;
    - поле Описание = поле Название + поле Дата начала.
